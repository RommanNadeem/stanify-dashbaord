# Stanify Dashboard

Social comment analytics dashboard (Overview, Inbox, Automation, Spam & Crisis, Engagement).

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Deploy to Vercel

1. Push this repo to GitHub.
2. In [Vercel](https://vercel.com), import the repo.
3. Vercel will detect Vite and use `npm run build` + `dist` automatically.
4. Deploy.
